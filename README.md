A fun spinning wheel game that predicts the NBA Championship winner! The odds are updated live based on standings and team performances. 

### How to Run

#### Option 1: Python Server (Recommended)
1. Make sure you have Python installed
2. Run the server script:
   ```bash
   python server.py
   ```
3. Your browser should automatically open to the game
4. If not, navigate to: `http://localhost:8000`

#### Option 2: Direct HTML File
1. Simply open `index.html` in your web browser
2. Double-click the file or right-click and select "Open with" ‚Üí your browser

### How to Play

1. Click the **"SPIN THE WHEEL"** button
2. Watch the wheel spin with all 30 NBA teams

### Files

- `index.html` - Main HTML structure
- `style.css` - Styling and animations
- `script.js` - Wheel logic and rigged spin functionality
- `server.py` - Simple Python HTTP server to run the app
